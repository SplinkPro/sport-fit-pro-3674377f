# Pratibha by SPLINK

AI-powered athlete assessment and development platform built on SAI research and Indian benchmarks.

## Stack
- Frontend: Vite + React 18 + TypeScript
- UI: Tailwind CSS + Radix UI / shadcn-ui
- State / data: TanStack Query, Supabase
- ML: TensorFlow.js (MoveNet pose detection)
- Testing: Vitest, Playwright

## Local development
bun install
bun run dev

## Build
bun run build

## Tests
bun run test
